# CKO Kickboxing Emerson Website

A modern, responsive website for CKO Kickboxing Emerson gym located at 500 Kinderkamack Rd, Emerson, NJ 07630.

## Pages

- **Home** (`index.html`) – Hero section, programs, reviews, FAQ
- **About** (`about.html`) – Gym story, coaches
- **Group Training** (`group-training.html`) – Group fitness / kickboxing classes
- **Personal Training** (`personal-training.html`) – 1-on-1 training info
- **Membership** (`membership.html`) – Membership plans
- **Contact** (`contact.html`) – Contact form, hours, map
- **Schedule** (`schedule.html`) – Class schedule (members only)
- **Members** (`members.html`) – Member login
- **Members Area** (`members-area.html`) – Portal with schedule & resources
- **Nutrition** (`nutrition.html`) – CKO Emerson cookbook
- **Tracker** (`tracker.html`) – Workout tracker

## Tech Stack

- Pure HTML, CSS, JavaScript (no frameworks)
- Firebase (authentication/storage)
- Google Fonts (League Spartan + Open Sans)
- Font Awesome 6 icons

## Deployment

Static hosting via Cloudflare Pages, Netlify, or similar at `ckoemerson.com`.
